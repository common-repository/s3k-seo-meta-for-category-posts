== Description ==
Add Custom SEO Meta tags for Wordpress posts and categories without any hassle and easy to use. This is a cool plugin that makes it easier to add SEO meta.


== Installation == 
<p>This section describes how to install the plugin and get it working.</p>
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Categories -> Edit/Update Category
4. Update SEO Meta Title, Keyword and Description for Categories
5. Go to Posts -> Edit/Update  Post
6. Update SEO Meta Title, Keyword and Description for posts
